Toast Message


Este componente permite apresentar mensagens de erro, aviso, sucesso ou informativas numa area sempre visível para o utilizador.

O componente permite a configuração de:

Mensagem a mostrar
Tipo de mensagem (erro, aviso, sucesso e informação)
Tempo de visualização
Fecha sem ou com intervenção do utilizador
'Path' para a pasta das imagens
A côr da mensagem de erro
A côr da mensagem de aviso
A côr da mensagem de sucesso
A côr da mensagem de informação
A percentagem do tamanho da mensagem em relação ao tamanho da area visível da janela
O 'jQuery selector' para o 'div' que vai conter as mensagens
A familia da fonte a usar nas mensagens
O tipo da fonte a usar nas mensagens
O tamanho da fonte a usar nas mensagens
A côr da fonte a usar nas mensagens
Posição das mensagens
A côr do 'border' das mensagens
A opacidade das mensagens
A velocidade com que as mensagens aparecem e desaparecem

Inicializar o Toast Message


Carregue o ficheiro toast-message.js com o toastMessage na main page da aplicação (depois de carregar o jQuery).
O componente necessita de jQuery para funcionar.
<script type="text/javascript" src="js/jquery.js"></script>
<script type="text/javascript" src="js/toast-message.js"></script>

Copie os ficheiros de imagem para a sua pasta de imagens.
toast-close.png
toast-paperclip.png

No javascript da main page inicialize e configure o Toast Message.
Os valores dos parametros imagePath e parentJQuerySelector devem ser alterados para se adequarem às suas necessidades.
var myToast = new toastMessage();
myToast.start(params);

Após inicializar o Toast Message pode começar a usar-lo.
myToast.showInfo("Bem vindo ao Toast Message");

Parametros


Podem ser configurados apenas os parametros que se desejar, nenhum dos parametros é obrigatório.
Para os parametros que não forem passados são usados os valores de 'default'.
Os parametros podem ser passados na inicialização, para configuração de 'defaults' ou na criação de uma mensagem.

myToast.start(params);
myToast.config(params);
myToast.show(params);

Os parametros são passados através de um objecto json com as seguintes propriedades:

Propriedade Tipo de dados Valor de defeito Valores admitidos Help
msg string empty string qualquer texto mensagem a mostrar
type string info error, warning, success, info tipo de mensagem
time int 10000 qualquer numero inteiro tempo que a mensagem fica visivel (milisegundos)
autoclose boolean true true, false a mensagem fecha sem intervenção do utilizador
imagePath string /img qualquer texto path para a pasta de imagens
errorColor string #ff0000 css color code côr das mensagens de erro
warningColor string #ff6600 css color code côr das mensagens de aviso
successColor string #009933 css color code côr das mensagens de sucesso
infoColor string #0066ff css color code côr das mensagens de informação
percentSize int 30 qualquer numero inteiro menor que 101 tamanho da mensagem em relação ao tamanho da janela (percentagem)
parentJQuerySelector string .container jquery selector jquery selector para o div que vai conter as mensagens
fontFamily string verdana times, courier, arial, verdana, etc familia da fonte
fontWeight string bold normal, bold, bolder, lighter, etc grossura da fonte
fontSize string 12px css size value tamanho da fonte
fontColor string #cccccc css color code côr da fonte
messagePosition string topright topright, topleft, bottomright, bottomleft posição das mensagens no ecrã
borderColor string #1a1a1a css color code côr do 'border' das mensagens
opacity int 85 qualquer numero inteiro menor que 101 opacidade das mensagens (percentagem)
fadeSpeed int 500 qualquer numero inteiro velocidade com que as mensagens aparecem e desaparecem (milisegundos)

Parametrização


myToast.start(params);

Ao inicializar o Toast Message deve configurar todos os parametros cujo valor de defeito não se adequa às suas necessidades.
Os valores dos parametros imagePath e parentJQuerySelector devem ser alterados para se adequarem às suas necessidades.

myToast.config(params);

Após inicializar o Toast Message pode sempre alterar as configurações.

myToast.show(params);

Mostra uma mensagem com as configurações passadas.
Não altera os valores de defeito, para alterar os valores de defeito use myToast.config(params).
São usados os valores de defeito para as propriedades messagePosition, percentSize e parentJQuerySelector.

Altere os parametros (ou não) e teste myToast.config(params) ou myToast.show(params) com os parametros definidos por si.
Do lado direito dos 'inputs' estão os valores de defeito correntes.

var params = {
msg:

type: info

time: 10000

autoclose: true

imagePath: /img

errorColor: #ff0000

warningColor: #ff6600

successColor: #009933

infoColor: #0066ff

percentSize: 30

parentJQuerySelector: .container

fontFamily: verdana

fontWeight: bold

fontSize: 12px

fontColor: #cccccc

messagePosition: topright

borderColor: #1a1a1a

opacity: 85

fadeSpeed: 500
};

myToast.config(params);

myToast.show(params);



Mensagens que se fecham sem intervenção do utilizador

Para evitar que uma mensagem se feche automaticamente pressione o 'clip'.

showError(msg) Mostra uma mensagem de erro que se fecha sem intervenção do utilizador.
myToast.showError(" ");

showWarning(msg) Mostra uma mensagem de aviso que se fecha sem intervenção do utilizador.
myToast.showWarning(" ");

showSuccess(msg) Mostra uma mensagem de sucesso que se fecha sem intervenção do utilizador.
myToast.showSuccess(" ");

showInfo(msg) Mostra uma mensagem de informação que se fecha sem intervenção do utilizador.
myToast.showInfo(" ");

Mensagens que só se fecham com intervenção do utilizador

Para que uma mensagem se feche automaticamente pressione o 'clip'.

showFixedError(msg) Mostra uma mensagem de erro que só se fecha com intervenção do utilizador.
myToast.showFixedError(" ");

showFixedWarning(msg) Mostra uma mensagem de aviso que só se fecha com intervenção do utilizador.
myToast.showFixedWarning(" ");

showFixedSuccess(msg) Mostra uma mensagem de sucesso que só se fecha com intervenção do utilizador.
myToast.showFixedSuccess(" ");

showFixedInfo(msg) Mostra uma mensagem de informação que só se fecha com intervenção do utilizador.
myToast.showFixedInfo(" ");
